Introduccin

Este cuaderno rene la teora y simulaciones prcticas de cuatro distribuciones discretas fundamentales: uniforme discreta, Bernoulli, binomial e hipergeomtrica. Para cada distribucin revisamos la funcin de masa de probabilidad (pmf), el valor esperado y la varianza, y luego ejecutamos dos o tres simulaciones breves que muestran cmo se comportan en la prctica las muestras aleatorias. Todas las simulaciones se basan en los generadores de nmeros aleatorios de R base (sample(), rbinom() y rhyper()), y cada una incluye visualizaciones que comparan las frecuencias relativas simuladas con las probabilidades tericas correspondientes.

Distribucin uniforme discreta

Teora

La distribucin uniforme discreta asigna la misma probabilidad a cada elemento de un conjunto finito. Si \(X\) toma valores en \({a, a+1, \ldots, b}\) con \(n = b - a + 1\) resultados posibles, entonces

\[ P(X = x) = \frac{1}{n}, \qquad x \in {a, \ldots, b}. \]

Su esperanza y varianza son

\[ \mathbb{E}[X] = \frac{a + b}{2}, \qquad \operatorname{Var}(X) = \frac{n^2 - 1}{12}. \]

Este modelo aplica cuando se selecciona un elemento de manera uniforme al azar, como al lanzar un dado o elegir un da al azar dentro de una lnea de tiempo corta.

Ejemplo 1: lanzar un dado justo

Generamos \(10000\) muestras de un dado justo de seis caras y comparamos las frecuencias simuladas con la probabilidad terica de \(1/6\).

n_rolls <- 10000
die_rolls <- sample(1:6, size = n_rolls, replace = TRUE)
die_distribution <- tibble(face = factor(die_rolls, levels = 1:6)) %>%
  count(face) %>%
  mutate(relative_freq = n / sum(n))

Distribucin de Bernoulli

Teora

La distribucin de Bernoulli describe un nico ensayo con dos posibles resultados: xito (1) con probabilidad \(p\) y fracaso (0) con probabilidad \(1 - p\). Su pmf es

\[ P(X = x) = p^x (1 - p)^{1 - x}, \qquad x \in {0, 1}. \]

La esperanza y la varianza son

\[ \mathbb{E}[X] = p, \qquad \operatorname{Var}(X) = p(1 - p). \]

Cualquier decisin de s/no con una probabilidad de xito fija puede modelarse como Bernoulli.

Ejemplo 1: Prueba de aprobacin/fallo de un componente

Supongamos que un componente fabricado aprueba la inspeccin con probabilidad (p = 0.92). Simulamos (2000) pruebas para ver con qu frecuencia el componente aprueba.

n_tests <- 2000
pass_prob <- 0.92
inspection_results <- rbinom(n = n_tests, size = 1, prob = pass_prob)
component_summary <- tibble(
  sample_mean = mean(inspection_results),
  sample_variance = var(inspection_results),
  theoretical_mean = pass_prob,
  theoretical_variance = pass_prob * (1 - pass_prob)
)
component_summary
## # A tibble: 1  4
##   sample_mean sample_variance theoretical_mean theoretical_variance
##         <dbl>           <dbl>            <dbl>                <dbl>
## 1       0.916          0.0774             0.92               0.0736
inspection_distribution <- tibble(result = factor(inspection_results, levels = c(0, 1), labels = c("Fail", "Pass"))) %>%
  count(result) %>%
  mutate(relative_freq = n / sum(n))

ggplot(inspection_distribution, aes(result, relative_freq)) +
  geom_col(fill = "#F28E2B") +
  labs(
    title = "Resultados de Inspeccin para un Ensayo de Bernoulli",

  )

Distribucin binomial

Teora

La distribucin binomial cuenta el nmero de xitos en \(n\) ensayos de Bernoulli independientes, cada uno con probabilidad \(p\) de xito. Para \(k = 0, 1, \ldots, n\), la pmf es

\[ P(X = k) = \binom{n}{k} p^k (1 - p)^{n - k}. \]

La esperanza y la varianza se derivan directamente de la suma de variables de Bernoulli independientes:

\[ \mathbb{E}[X] = np, \qquad \operatorname{Var}(X) = np(1 - p). \]

Aplicaciones tpicas incluyen contar el nmero de artculos defectuosos en un lote cuando los defectos son raros e independientes, o contabilizar xitos en juegos repetidos.

Ejemplo 1: tiros libres en baloncesto

Un jugador realiza (n = 10) tiros libres con probabilidad de acierto (p = 0.75). Repetimos la serie de 10 tiros (5000) veces y comparamos los conteos empricos con la pmf binomial terica.

basketball_distribution <- tibble(made = made_shots) %>%
  count(made) %>%
  mutate(relative_freq = n / sum(n),

Distribucin hipergeomtrica

Teora

La distribucin hipergeomtrica mide el nmero de xitos en \(n\) extracciones de una poblacin finita sin reemplazo. Si una poblacin tiene \(N\) elementos con \(K\) xitos y \(N - K\) fracasos, y extraemos \(n\) elementos, entonces para \(k\) xitos la pmf es

\[ P(X = k) = \frac{\binom{K}{k} \binom{N - K}{n - k}}{\binom{N}{n}}, \qquad \max(0,, n - (N - K)) \le k \le \min(n,, K). \]

La esperanza y la varianza incorporan la correccin por poblacin finita:

\[ \mathbb{E}[X] = n \frac{K}{N}, \qquad \operatorname{Var}(X) = n \frac{K}{N} \left(1 - \frac{K}{N}\right) \frac{N - n}{N - 1}. \]

Esta distribucin es adecuada cuando las selecciones se realizan sin reemplazo, como al sacar cartas de una baraja o muestrear unidades de un lote.

Ejemplo 1: sacar ases de una baraja

Extraemos cinco cartas de una baraja estndar de 52 cartas. El nmero de ases en la muestra sigue una distribucin hipergeomtrica con (N = 52), (K = 4) y (n = 5).

n_samples <- 7000
num_aces <- rhyper(nn = n_samples, m = 4, n = 52 - 4, k = 5)

aces_distribution <- tibble(aces = num_aces) %>%
  count(aces) %>%
  mutate(relative_freq = n / sum(n),
         theoretical = dhyper(aces, m = 4, n = 48, k = 5))

Conclusiones

En estos ejemplos, las frecuencias simuladas se alinean estrechamente con las pmf tericas de cada distribucin una vez que reunimos una muestra suficientemente grande. Las distribuciones uniforme y de Bernoulli tratan ensayos individuales, mientras que las distribuciones binomial e hipergeomtrica se extienden al conteo de xitos a lo largo de mltiples extraccionescon y sin reemplazo, respectivamente. Las simulaciones ilustran cmo la variacin aleatoria se manifiesta en muestras finitas y proporcionan una verificacin emprica de los modelos tericos.
